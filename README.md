# Windows 98 Gravity OS (星律：引力编织者 - 复古版)

![Windows 98 Style](https://img.shields.io/badge/Style-Windows_98-008080?style=flat-square)
![Game Engine](https://img.shields.io/badge/Engine-Canvas_API-orange?style=flat-square)

**Windows 98 Gravity OS** 是一款基于 Web 的重力合成游戏，运行在一个高度还原的 Windows 98 桌面模拟环境中。它将现代物理引擎的爽快感与 90 年代的复古美学完美融合，提供了一种独特的、既硬核又治愈的交互体验。

## 🎮 游戏核心 (Core Gameplay)

这不仅仅是一个 UI 皮肤，而是一个完整的操作系统模拟环境下的物理游戏：

* **引力机制**：在屏幕任意位置按住鼠标（或触摸），创造一个“引力黑洞”，吸引周围漂浮的球体。
* **合成升级**：引导相同颜色的球体相撞，它们会合并成更高级的球体，并播放悦耳的合成音效。
* **物理模拟**：包含惯性、摩擦力、墙壁反弹等真实物理特性，手感丝滑。

## ✨ 特色功能 (Features)

### 🖥️ 沉浸式复古体验
* **CRT 滤镜**：屏幕覆盖扫描线效果，模拟老式大屁股显示器的质感。
* **Win98 UI**：经典的青色桌面、灰色斜面按钮 (Bevel)、像素字体和蓝紫渐变标题栏。
* **无图源设计**：所有图标（如我的电脑、回收站）均使用 SVG 代码手绘，无需加载外部图片资源，永不破图。

### ⚙️ 进阶玩法系统
1.  **💣 蓝屏炸弹 (BSOD Box)**：
    * 随机生成的蓝色错误方块（Error 0E）。
    * **开关功能**：可在游戏内菜单自由开启或关闭。
    * **效果**：撞击会触发“系统崩溃”清屏特效，瞬间消除周围球体并获得大量分数。
2.  **⚡ 速度调节**：
    * 支持 1x 到 5x 速度切换，挑战你的动态视力与反应极限。
3.  **🔊 程序化音频引擎 (Audio Engine)**：
    * 使用 Web Audio API 实时合成音效，无需 MP3 文件。
    * **智能防爆音**：内置音频节流阀（Throttling），防止在发生连锁爆炸时产生刺耳噪音，保护你的耳朵。

### 🥚 彩蛋与交互
* **网上邻居**：模拟真实的 56k 调制解调器拨号过程（含握手噪音），连接到虚假的“局域网排行榜”。
* **回收站**：点击会有随机生成的幽默错误提示（Trash Talk），致敬那个电脑经常报错的年代。
* **系统重启**：点击“重启”按钮会触发真实的 BIOS 自检动画（内存检测、硬件扫描），这是清空分数的最硬核方式。

## 🕹️ 操作指南 (Controls)

| 动作 | 效果 |
| --- | --- |
| **长按 (鼠标左键 / 触摸)** | 在光标处产生引力，吸引球体 |
| **松开** | 停止引力，球体随惯性漂浮 |
| **双击桌面图标** | 打开对应程序 (如 Gravity.exe, 网上邻居) |
| **点击音效按钮** | 开启/关闭声音 (推荐开启以获得最佳体验) |
| **点击重启按钮** | 模拟 BIOS 重启并重置游戏进度 |

## 🚀 安装与运行 (Installation)

本项目为**单文件 (Single-file)** 架构，无需复杂的构建过程，即开即用。

1.  下载 `index.html` 文件。
2.  使用任意现代浏览器 (Chrome, Edge, Firefox, Safari) 打开该文件。
3.  **注意**：由于现代浏览器的自动播放策略，进入页面后请先点击游戏窗口内的“音效”按钮以激活音频引擎。

## 🛠️ 技术栈 (Tech Stack)

* **HTML5 Canvas**: 用于高性能的游戏渲染（目标 60fps）。
* **Vanilla JavaScript**: 无任何第三方库依赖 (No jQuery, No React)，纯原生 JS 实现。
* **Web Audio API**: 用于生成复古的合成器音效（Square/Triangle/Sawtooth waves）。
* **CSS3**: 实现 Windows 98 的 UI 样式、模态窗口管理和 CRT 滤镜效果。
